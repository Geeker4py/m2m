# M2M

> Markdown to More

将Markdown转换博客、网站，也可以生成一封HTML格式的邮件并发送。它还能将你的博客或网站自动提交到git，或者发布到你的服务器上。

## 如何安装

`sudo gem install m2m --source http://mirrors.aliyun.com/rubygems/ -n /usr/local/bin`
